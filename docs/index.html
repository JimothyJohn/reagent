Deploy an AI agent to Microsoft Teams in minutes, powered by OpenClaw.

Prerequisites +

Make sure you have the following before starting:

  • Azure subscription + Azure CLI installed and logged in (az login)
  • SSH key pair — run ssh-keygen if you don't have one at ~/.ssh/id_rsa.pub
  • Anthropic API key — get one from console.anthropic.com
  • Python 3.12+ and uv (Python package manager)
  • Microsoft 365 Business or Education account
Important: Personal and free Microsoft Teams accounts do NOT support custom app sideloading. You need a Microsoft 365 Business Basic (or higher) or Education plan to upload custom bots.
Clone & configure + 
git clone https://github.com/nkuhn-vmw/reagent.git
cd reagent

That's it for now — the automated setup in the next step will generate your .env file with all required credentials.

Run automated setup +

A single command provisions all Azure infrastructure and configures the bot end-to-end:

uv run python main.py setup

You'll be prompted for your Anthropic API key if it isn't already in .env. The command handles everything else automatically:

  • Creates Azure resource group, VM, and networking (NSG rules)
  • Registers an Azure AD app + service principal + bot service
  • Installs Node.js, OpenClaw, and nginx on the VM
  • Configures HTTPS with Let's Encrypt (TLS is required by Teams)
  • Deploys the OpenClaw gateway configuration
  • Writes all credentials to .env
HTTPS is mandatory. Microsoft Teams requires a valid TLS certificate on your bot endpoint. The setup script handles this automatically using Let's Encrypt and nginx as a reverse proxy.

After setup completes, you can verify the deployment:

uv run python main.py status
uv run python main.py logs
Enable custom app sideloading +

Your Teams tenant must allow custom app uploads. A Teams admin needs to enable this:

  1. Go to admin.teams.microsoft.com
  2. Navigate to Teams apps → Setup policies → Global (Org-wide default)
  3. Toggle Upload custom apps to On
  4. Save and wait a few minutes for the policy to propagate
Propagation delay: Policy changes can take up to 24 hours to take effect, though it usually happens within a few minutes.
Package & sideload the Teams app +

Build the Teams app manifest zip (this injects your App ID from .env):

./scripts/package-teams-app.sh

Then upload it to Teams:

  1. Open Microsoft Teams
  2. Click Apps in the sidebar
  3. Click Manage your apps → Upload a custom app
  4. Select the generated reagent-teams-app.zip file
Chat with the bot +

Find Reagent in your Teams app list. You can:

  • Send it a direct message for 1:1 conversation
  • Add it to a channel so your whole team can interact with it

The bot connects to Claude via your Anthropic API key. To redeploy after config changes:

uv run python main.py deploy
