Clone & configure + 
git clone https://github.com/nkuhn-vmw/reagent.git
cd reagent
cp .env.example .env

Open .env and fill in your Azure credentials (see Step 2 if you don't have them yet).

Create Azure resources +

Create a resource group, a B1s Linux VM, and an Azure Bot. This is the one-time infrastructure step.

# Resource group
az group create --name reagent-rg --location eastus

# Linux VM (Ubuntu 24.04, ~$7.59/mo)
az vm create \
  --resource-group reagent-rg \
  --name reagent-vm \
  --image Ubuntu2404 \
  --size Standard_B1s \
  --admin-username reagent \
  --generate-ssh-keys

# Open bot port
az network nsg rule create \
  --resource-group reagent-rg \
  --nsg-name reagent-vmNSG \
  --name allow-bot \
  --priority 1020 \
  --destination-port-ranges 3978 \
  --protocol Tcp --access Allow --direction Inbound

# Register Azure AD app
az ad app create --display-name "reagent-teams-bot" \
  --sign-in-audience AzureADMyOrg

# Create secret (save the password!)
az ad app credential reset --id <APP_ID>

# Create bot service (free tier)
az bot create \
  --resource-group reagent-rg \
  --name reagent-teams-bot \
  --app-type SingleTenant \
  --appid <APP_ID> \
  --tenant-id <TENANT_ID> \
  --endpoint "https://<VM_DNS>:3978/api/messages"

# Enable Teams channel
az bot msteams create \
  --resource-group reagent-rg \
  --name reagent-teams-bot
Install OpenClaw on your VM +

SSH into the VM and install Node.js + OpenClaw globally.

ssh reagent@<VM_IP>

# Install Node.js 22
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install OpenClaw
sudo npm install -g openclaw@latest
Deploy config & start gateway +

Use the built-in CLI to push your config (with secrets from .env) to the VM and restart.

python main.py deploy

Or check status / tail logs:

python main.py status
python main.py logs
Sideload into Teams +

Package the Teams manifest and upload it.

# Build the zip (substitutes your App ID from .env)
./scripts/package-teams-app.sh

Then in Microsoft Teams:

Apps → Manage your apps → Upload a custom app → select reagent-teams-app.zip

Find the Reagent bot in your app list, add it to a channel or DM it directly.
